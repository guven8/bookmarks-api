## Installation

```bash
$ npm install
```

## Initialization

```bash
# start DB
$ npm run db:dev:up

# run migrations
$ npm run prisma:dev:deploy
```

## Running the app

```bash
App runs on localhost:3333

# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Test

```bash
# e2e tests
$ npm run test:e2e
```
